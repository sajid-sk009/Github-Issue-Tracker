Question-01:What is the difference between var, let, and const?

Ans: var,let and const all are use for declear variable in js. Since there task is same but there characteristics and working process is different. The difference is:
var can be redecleared and it is known as functional scoped.

let can not be redecleared but it can reassigned and it is known as block scope.

const is also known as block scope but in const redecleared or reassigned is not possible.

Question-03: What is the difference between map(), filter(), and forEach()?

Ans: All of these are work on array. But there working output is different. Like as 

map used to transform  data. It leaving the original array unchange it generates a new array by applying a designated function to every element of the original array.

filter recived an array and give the data which are passed the given condition. It is not change the original array.

foreach applies the similar code in each element. It applies the condition only but it is not return anything. It is not change the original array.
